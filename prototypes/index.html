Redesign prototypes

Three from-scratch design directions for ukrainewarlosses.com, all powered by the same russian-losses.json dataset. Every prototype has a light/dark theme toggle, 5 languages (EN, FR, DE, UK, RU), interactive charts, and a credible-resources section. Append ?theme=light|dark&lang=en|fr|de|uk|ru to any URL to deep-link a variant.

Prototype A · dark-first

Command Center

An operations-dashboard feel: hero counters with live deltas, a card grid with sparklines for all 14 categories, and a large explorable chart. Optimized for daily returning visitors who want everything at a glance.

Retention hook: today / 7-day / 30-day deltas everywhere → always something new. Prototype B · light-first

The Briefing

Data-journalism style: a serif masthead and written lede with inline figures, a clean "daily report" table, milestone timeline, and one focused chart. Familiar and highly readable — newspaper conventions any age group navigates instinctively.

Acquisition hook: reads like an article → naturally quotable and linkable. Prototype C · dark-first

The Ledger

A stark, typography-first tally: one giant animated counter, a strict grid of the full ledger, and minimal chrome. Poster-like and instantly screenshotable — built for word-of-mouth sharing.

WOM hook: the hero section is the share card.

All prototypes are static (no build step, no dependencies) and load the dataset at ../../russian-losses.json. Serve the repo root with any static server, e.g. python3 -m http.server, then open /prototypes/.
